🚀 Phương pháp 1: Quick Start (Khuyến nghị)

Chạy test và auto-fix trong một lệnh duy nhất:

php quick_test.php

Hoặc click vào nút bên dưới để chạy từ web:

▶️ Run Quick Test

🐳 Phương pháp 2: Docker (Production)

Chạy tests trong môi trường Docker isolated:

run_docker_tests.bat

Docker sẽ tự động:

  • Khởi động MySQL test database
  • Chạy automated tests
  • Áp dụng auto-fixes nếu cần
  • Chạy health monitor
  • Dọn dẹp môi trường

🌐 Phương pháp 3: Web Dashboard

Quản lý tests qua giao diện web trực quan:

📊 Open Dashboard

Dashboard cho phép:

  • Xem kết quả tests real-time
  • Chạy tests với một click
  • Áp dụng fixes tự động
  • Xem health status
  • Xem lịch sử tests

⚙️ Phương pháp 4: Manual (Advanced)

Chạy từng bước riêng biệt:

# Bước 1: Chạy tests
php automated_test_runner.php

# Bước 2: Áp dụng fixes (nếu có lỗi)
php auto_fix_runner.php

# Bước 3: Chạy health monitor
php health_monitor.php

🎯 Bắt đầu ngay
