## 🔗 Tích Hợp Với Thanh Toán

### 1. Luồng Tính Phí
```
Khách chọn địa chỉ
    ↓
Hệ thống xác định tỉnh/thành
    ↓
Tính trọng lượng đơn hàng
    ↓
Query shipping_fees (theo priority)
    ↓
Áp dụng công thức: base_fee + (weight - weight_from) × fee_per_kg
    ↓
Kiểm tra miễn phí ship (order_value >= min_order_free_ship)
    ↓
Hiển thị phí cuối cùng
```

### 2. Công Thức Tính Phí

```php
function calculateShippingFee($provinceId, $weight, $orderValue) {
    // 1. Tìm cấu hình phí phù hợp (theo priority)
    $fee = findShippingFee($provinceId, $weight);
    
    // 2. Tính phí cơ bản
    $baseFee = $fee['base_fee'];
    
    // 3. Tính phí theo trọng lượng
    $extraWeight = max(0, $weight - $fee['weight_from']);
    $weightFee = $extraWeight * $fee['fee_per_kg'];
    
    // 4. Tổng phí
    $totalFee = $baseFee + $weightFee;
    
    // 5. Kiểm tra miễn phí ship
    if ($orderValue >= $fee['min_order_free_ship']) {
        return 0; // Miễn phí
    }
    
    return $totalFee;
}
```

## ⚠️ Lưu Ý Quan Trọng

### 1. Priority (Độ Ưu Tiên)
```
10 = Cao nhất (Nội thành HCM)
9  = Rất cao (Nhanh nội thành)
8  = Cao (Ngoại thành HCM)
7  = Trung bình cao (Tỉnh lân cận)
6  = Trung bình (Đồng bằng)
5  = Trung bình thấp (Miền Trung)
4  = Thấp (Miền Bắc)
1  = Thấp nhất (Mặc định)
```

**Nguyên tắc:** Số cao hơn = ưu tiên áp dụng trước

### 2. Khoảng Trọng Lượng
- Không được overlap (chồng lấn)
- Ví dụ đúng: 0-3kg, 3-10kg, >10kg
